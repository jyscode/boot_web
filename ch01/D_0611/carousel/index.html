HTML(HyperText Markup Language)


HTML(HyperText Markup Language)은 웹 페이지의 구조와 내용을 정의하는 마크업 언어입니다. HTML은 태그(tag)를 사용하여 텍스트, 이미지, 링크, 폼 등 다양한 요소를 웹 페이지에 추가하고, 이들 요소의 계층 구조를 정의합니다. HTML 문서는 브라우저에 의해 해석되어 사용자가 볼 수 있는 웹 페이지로 렌더링됩니다.

CSS(Cascading Style Sheets)


CSS 는 웹 페이지의 레이아웃, 디자인 및 표현을 제어하기 위해 사용되는 스타일시트 언어입니다.CSS는 HTML 문서의 구조를 스타일링하고, 색상, 폰트, 레이아웃 등 다양한 요소의 시각적 표현을 정의합니다. CSS는 웹 디자인의 핵심 요소 중 하나로, 사용자 경험과 웹 페이지의 접근성을 향상시키는 데 중요한 역할을 합니다.

JavaScript


JavaScript는 웹 페이지를 동적으로 만들기 위해 사용되는 고수준의, 다중 패러다임 프로그래밍 언어입니다. 자바스크립트는 웹 개발의 핵심 기술 중 하나로, HTML과 CSS와 함께 사용되어 웹 페이지의 구조와 스타일을 정의하고, 자바스크립트는 웹 페이지의 동작을 제어합니다. 자바스크립트는 클라이언트 측에서 주로 사용되지만, 서버 측에서도 Node.js와 같은 런타임 환경을 통해 사용됩니다.
